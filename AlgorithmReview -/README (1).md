# Algorithm Review

## Case Study 1 : Array

- คำนวณคะแนนรวม
- คะแนนเฉลี่ย
- คะแนนสูงสุด
- คะแนนต่ำสุด
- จำนวนนักศึกษาที่ได้คะแนนตั้งแต่ 7 คะแนนขึ้นไป
- รายชื่อนักศึกษาที่ได้คะแนนต่ำกว่า 5

Time Complexity : O(n)

## Case Study 2 : Stack

ใช้ Stack<String>

ดำเนินการ

- push()
- pop()
- peek()
- isEmpty()

หลักการทำงาน

LIFO (Last In First Out)

Time Complexity

- push O(1)
- pop O(1)
- peek O(1)
